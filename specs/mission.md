# Mission

The purpose of this project is to create an interactive, single-page web application to host Python exercises for university computer science students. It aims to provide a clean, engaging "modern development environment" aesthetic, ensuring the interface is lightweight and highly performant.

## Target Audience

University computer science students participating in a Python programming lab or self-study.

## Key Objectives

- Host 20 exercises covering sets, dictionaries, classes, and trees.
- Provide a responsive and modular UI using native Web Components.
- Implement an interactive ASCII particle background using Three.js that reacts with a neon pink accent color on mouse hover.
- Ensure the application is easily deployable to Vercel.
